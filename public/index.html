Get Started

  1. Go to configs and configurate your robot
  2. Go to the Blocks tab to code with a block-style language, or go to JS for Javascript!
  3. Either tabs should have starter codes. Feel free to play around with it.
  4. Run and test your bot by clicking "Run robot"!
  5. Test your skills by playing with others by clicking "Play a Match"

Helpful tips

  • Make sure you run your robot if you want to save it.
  • If you want to run the robot with your Javascript code, make sure your JS tab is open when you press "Run Robot". Same thing with Blocks.
  • If you are programming with Blocks look in the Actions and Sensors tabs for robot-specific code blocks!
